Getting Started

AI Film Studio is a cloud-native platform that transforms text scripts into cinematic short films using AI-powered image/video generation. This guide will help you set up your own instance.

Prerequisites

Cloud & Infrastructure

  • AWS Account with appropriate permissions
  • Terraform >= 1.5
  • Docker Desktop
  • kubectl (for EKS)

Development

  • Node.js >= 18
  • Python >= 3.11
  • GitHub account for CI/CD

Installation

1. Clone the Repository

git clone https://github.com/AI-Empower-HQ-360/AI-Film-Studio.git
cd AI-Film-Studio

2. Set Up Backend

cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env  # Configure your environment variables
uvicorn src.main:app --reload

3. Set Up Worker

cd worker
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/main.py

4. Set Up Frontend

cd frontend
npm install
cp .env.example .env.local
npm run dev

API Reference

The AI Film Studio API is built with FastAPI and provides RESTful endpoints for all platform functionality.

Base URL: https://api.your-domain.com/api/v1
Authentication: Bearer Token (JWT)
Content-Type: application/json

API Endpoints

Authentication

POST /auth/register

Register a new user account

{
  "email": "user@example.com",
  "password": "SecurePass123"
}
POST /auth/login

Authenticate and receive JWT tokens

{
  "email": "user@example.com",
  "password": "SecurePass123"
}

Projects

POST /projects

Create a new film project

{
  "title": "My First Film",
  "script": "A hero embarks on an epic journey..."
}
GET /projects

List all user projects with pagination

Film Generation

POST /projects/{projectId}/generate

Submit a film generation job

GET /jobs/{jobId}

Track job progress and status

GET /jobs/{jobId}/download

Get download URL for completed film

Health Check

GET /health

Check API health status

// Response
{
  "status": "healthy",
  "version": "0.1.0",
  "service": "AI Film Studio"
}

Infrastructure Deployment

Initialize Terraform

cd infrastructure/terraform/environments/dev
terraform init

Plan Infrastructure

terraform plan -out=tfplan

Apply Infrastructure

terraform apply tfplan

Environments

Environment Purpose Infrastructure
Dev Rapid development and testing Small instances, single AZ
Test/QA Integration and performance testing Mirrors prod, scaled down
Staging Pre-production validation Prod-like, blue-green ready
Production Live user traffic Multi-AZ, auto-scaling, HA
