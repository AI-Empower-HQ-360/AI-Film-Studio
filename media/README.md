# Media Storage

Temporary storage for media processing.

⚠️ **Note:** This directory is for local development only. Production uses AWS S3.

## Directory Structure

- `images/` - User-uploaded images
- `videos/` - Generated videos
- `thumbnails/` - Auto-generated thumbnails
- `subtitles/` - Subtitle files

## Gitignore

All media files are ignored by git (see .gitignore).
