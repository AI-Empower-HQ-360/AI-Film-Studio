# AI Film Studio Documentation

## Overview

AI Film Studio is an end-to-end AI-powered film production pipeline that transforms scripts into finished video content.

## Pipeline Stages

1. **Script** - Parse or generate film scripts
2. **Scenes** - Analyze scripts and break them into scenes
3. **Shots** - Generate individual shots using AI
4. **Video** - Assemble shots into a complete video
5. **Output** - Export the final film in various formats

## Getting Started

See the [examples](../examples/) directory for usage examples.

## Modules

- `ai_film_studio.script` - Script parsing and generation
- `ai_film_studio.scenes` - Scene analysis and breakdown
- `ai_film_studio.shots` - Shot generation and composition
- `ai_film_studio.video` - Video assembly and processing
- `ai_film_studio.output` - Final export and output
- `ai_film_studio.utils` - Configuration and utilities
