# AI Assistants Documentation

This folder contains documentation for setting up and using AI coding assistants with the AI Film Studio project.

## Contents

- **AI_ASSISTANTS_STATUS.md** - Current status of AI assistant integrations
- **AI_ASSISTANT_CONNECTION.md** - Connection and configuration guide
- **AI_CODING_ASSISTANTS_SETUP.md** - Complete setup instructions
- **CLAUDE_AI_INTEGRATION_GUIDE.md** - Claude AI specific integration guide
- **CLINE_AI_EXTENSION_GUIDE.md** - Cline VS Code extension guide
- **HOW_TO_USE_AI_ASSISTANT.md** - User guide for AI assistants
- **YOUR_AI_ASSISTANTS.md** - Overview of available AI assistants

## Supported AI Assistants

1. **GitHub Copilot** - Code completion and suggestions
2. **Claude AI** - Anthropic's AI assistant
3. **Cline** - VS Code AI extension
